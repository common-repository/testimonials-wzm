== Description ==

Testimonials is a brand new plugin that gives the user the option to create testimonials. These testimonials can be placed anywhere on any page with the shortcodes that come with the premium version. With the testimonials plugin you can manage how many testimonials you want to display per page. Premium also adds lots of new options to the settings page in order to change the testimonials to your needs.


#### Most powerful features:
* **User-friendly** - Anyone can edit the testimonials to their needs.
* **Totally integrated with WordPress** - media manager, post editor

#### Features:
* Create testimonials with an easy editor.
* Manage your testimonials in one easy page.
* Customize how many testimonials you want to display per page.
* Makes a testimonials page where they all will be displayed.
* Responsive testimonial layout

#### Testimonials Pro Features:
* Testimonial shortcodes.
* Sliders.
* Custom css option.
* 12 fonts.
* Option to include a google font.
* Title font option.
* Content font option.
* 5 slider transitions.
* Slider speed option.
* Current slide color option.
* Title color option.
* Content color option.
* Premium support.
